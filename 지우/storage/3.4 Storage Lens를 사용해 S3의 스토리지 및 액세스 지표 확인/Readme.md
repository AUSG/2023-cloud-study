# 3.4 Storage Lens를 사용해 S3의 스토리지 및 액세스 지표 확인

> _S3 사용 패턴을 관찰하기 위해 S3 observability 및 분석을 확인할 수 있는 Storage Lens를 구성한다_

- Console에서 작업!

① S3 콘솔 탐색 창에서 **[ S3 Storage Lens ]** 선택

② [ 대시보드 생성 ] 선택

- 이름 지정
- **[ 대시보드 범위 ]** → **[ 리전 및 버킷 포함 ]** 선택
- **[ 고급 지표 및 권장 사항 ]** 활성화

③ 내보내기 설정에서 기본값 선택

- 자동 내보내기

④ **[ 대시보드 만들기 ]** 클릭해서 대시보드 확인
